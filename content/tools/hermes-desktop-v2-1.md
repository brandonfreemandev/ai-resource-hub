### What's new

Hermes Desktop v2.1 focuses on memory efficiency — the headline is a ~40% reduction in RAM usage for
the same models, which brings 7B–8B class models comfortably onto a typical laptop.

### How to use

1. Install with Homebrew: `brew install hermes-desktop --stable`
2. Launch the app and pick a model from the built-in library.
3. Start chatting — everything runs on your machine, nothing leaves it.
