### Why beginners love it

Ollama hides all the painful parts of running local models. You install it, run `ollama run llama3`, and
you're talking to a model that lives entirely on your computer.

### Quick start

1. Install: `curl -fsSL https://ollama.com/install.sh | sh`
2. Pull and run a model: `ollama run llama3`
3. Or hit the local API at `http://localhost:11434` from your own apps.
