### Why it matters

If the command line is intimidating, LM Studio is the gentlest on-ramp to local AI: a clean GUI with a
built-in model browser and a one-click local server.

### Quick start

1. Download LM Studio for your OS.
2. Use the in-app search to find a model (start small, e.g. an 8B).
3. Click **Download**, then **Chat** — or start the local server to use it from other apps.
