### Why it matters

ComfyUI turns generative image/video workflows into a visual graph. Beginners can start from shared
templates and gradually understand each node instead of treating the model as a black box.

### Quick start

1. Clone: `git clone https://github.com/comfyanonymous/ComfyUI`
2. Install requirements and launch per the docs.
3. Load a starter workflow and tweak one node at a time.
