### Why it matters

Open WebUI wraps your local models in a familiar, feature-rich chat interface — conversation history,
saved prompts, and document Q&A — so local AI feels as good as the hosted tools.

### Quick start

1. Run with Docker: `docker run -d -p 3000:8080 ghcr.io/open-webui/open-webui:main`
2. Open the UI and connect it to your Ollama instance.
3. Start chatting, save prompts, and upload docs to chat with them.
